## Responsive Breakpoints

### Mobile: 320px - 767px
- Single column
- Bottom navigation
- Collapsible filters
- Swipe gestures

### Tablet: 768px - 1023px  
- 2-3 column grid
- Side navigation
- Persistent filters
- Touch optimized

### Desktop: 1024px+
- 4-6 column grid
- Top navigation
- Advanced filters
- Hover states
